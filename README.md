## 💼 Featured Projects

### 👓 DRISHTI — AI-Powered Smart Glasses

A wearable IoT device for the visually impaired, built on ESP32 and IR sensors for obstacle detection. Includes a Python AI bridge with computer vision for automated scene descriptions delivered via audio cues, paired with a Flutter app for real-time location tracking and guardian monitoring.

**Tech:** Flutter, Dart, ESP32, Python, Computer Vision, Firebase

---

### ❤️ DILPROTEK — AI-Integrated Health Platform

A heart-health monitoring system combining AI risk prediction with OCR-based medical report analysis. Backend built on Python Flask for low-latency ML processing, with a secure doctor-patient portal and an AI-driven food nutrition scanner.

**Tech:** Flutter, Python (Flask), Machine Learning, OCR, Firebase, REST APIs
